## 📖 Overview
**PolicySchield AI** is a high-performance browser auditor that dismantles complex legal jargon. It executes **real-time neural summarization** to provide users with a "Safety Signal" before they click "I Agree."

### 🚀 Key Innovations
- **Vedic-Futurist Design:** A UI that blends traditional Indian motifs with high-tech cyberpunk aesthetics.
- **Optimized for 16GB RAM:** Aggressive memory management for developer-grade machines.
- **Gemini 1.5 Flash Integration:** Utilizing the `v1beta` endpoint for sub-second summarization of 15,000+ characters.

---

## 🛠️ Technical Stack & Under the Hood
This extension is built on **Chrome Manifest V3** with a focus on asynchronous message passing and secure API handling.

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | HTML5 / CSS3 / JS | Cyber-Bharat Glassmorphism UI |
| **Logic** | Chrome Scripting API | DOM Scraping & Text Extraction |
| **AI Engine** | Google Gemini 1.5 Flash | LLM Summarization & Risk Scoring |
| **Storage** | SQLite / Supabase | Local Caching & Sync (Upcoming) |
